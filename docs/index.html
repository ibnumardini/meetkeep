MeetKeep GitHub
Chrome Extension

Keep track of time
in every meeting

MeetKeep automatically injects a meeting timer into Google Meet, so you always know how long you've been in a call.

Download View on GitHub

Install in 3 steps

1

Download the latest .zip from GitHub Releases and extract it.

2

Open Chrome and go to chrome://extensions. Enable Developer mode (top-right toggle).

3

Click Load unpacked and select the extracted folder. Done, open any Google Meet call.
